┌─────────────────────────────────────────────────────────────────────────────┐
│ 📌 Direct 방식 vs Ratio 방식 - 뭐가 다른가요?                               │
├─────────────────────────────────────────────────────────────────────────────┤
│                                                                             │
│ 🎯 Direct 방식 (직접 예측)                                                  │
│ ─────────────────────────────────────────────────────────────────────────── │
│ "점유율 자체를 바로 예측하는 방식"                                          │
│                                                                             │
│ 예시: 현재 점유율이 16%니까, 다음 달은 15.8%, 그 다음 달은 15.6%...         │
│       이렇게 점유율 숫자를 직접 예측합니다.                                  │
│                                                                             │
│ 문제점: 시장 전체가 얼마나 커지는지 고려하지 않음                            │
│         → GS가 충전기를 많이 설치해도 시장이 더 빨리 커지면 점유율 하락      │
│         → 이런 상황을 제대로 반영 못함                                       │
│                                                                             │
│ ─────────────────────────────────────────────────────────────────────────── │
│                                                                             │
│ 🎯 Ratio 방식 (비율 계산) ← 현재 시뮬레이터가 사용하는 방식                  │
│ ─────────────────────────────────────────────────────────────────────────── │
│ "GS 충전기 수와 시장 전체 충전기 수를 각각 예측한 후 비율로 계산"            │
│                                                                             │
│ 공식: 점유율 = (GS 충전기 수 ÷ 시장 전체 충전기 수) × 100                   │
│                                                                             │
│ 예시:                                                                       │
│   - GS 충전기: 현재 73,912대 → 6개월 후 74,500대 예측                       │
│   - 시장 전체: 현재 459,523대 → 6개월 후 497,000대 예측                     │
│   - 점유율 = 74,500 ÷ 497,000 × 100 = 14.99%                               │
│                                                                             │
│ 장점: 시장 성장 속도를 반영하여 더 현실적인 예측 가능                        │
│       → GS가 충전기를 추가 설치하면 시장 전체도 그만큼 커지는 것을 반영      │
│                                                                             │
│ ─────────────────────────────────────────────────────────────────────────── │
│                                                                             │
│ 📊 실제 비교 결과                                                           │
│   - Ratio 방식 오차율: 1.24% (100번 예측하면 평균 1.24번 틀림)               │
│   - Direct 방식 오차율: 1.46% (100번 예측하면 평균 1.46번 틀림)              │
│   → Ratio 방식이 약 15% 더 정확함!                                          │
│                                                                             │
└─────────────────────────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────────────────────────┐
│ 📌 백테스트가 뭔가요?                                                       │
├─────────────────────────────────────────────────────────────────────────────┤
│                                                                             │
│ "과거 데이터로 예측 모델을 시험해보는 것"                                    │
│                                                                             │
│ 예시:                                                                       │
│   1. 2025년 3월까지의 데이터만 가지고 있다고 가정                            │
│   2. 이 데이터로 2025년 4월, 5월, 6월 점유율을 예측                          │
│   3. 실제 4월, 5월, 6월 데이터와 비교                                        │
│   4. 얼마나 맞았는지 오차 계산                                               │
│                                                                             │
│ 이렇게 여러 시점에서 반복 테스트하면 모델의 실제 정확도를 알 수 있습니다.    │
│                                                                             │
└─────────────────────────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────────────────────────┐
│ 📌 주요 지표 쉬운 설명                                                      │
├─────────────────────────────────────────────────────────────────────────────┤
│                                                                             │
│ 📈 R² (결정계수) - "데이터를 얼마나 잘 설명하나?"                            │
│   - 0~1 사이 값 (1에 가까울수록 좋음)                                        │
│   - 0.95 = 데이터의 95%를 설명 가능 (매우 좋음)                              │
│   - 0.50 = 데이터의 50%만 설명 가능 (보통)                                   │
│                                                                             │
│ 📉 MAE (평균 절대 오차) - "평균적으로 얼마나 틀리나?"                        │
│   - 예측값과 실제값의 차이 평균                                              │
│   - MAE 0.17%p = 평균적으로 0.17%p 정도 틀림                                │
│   - 예: 실제 16%인데 15.83% 또는 16.17%로 예측                              │
│                                                                             │
│ 📊 MAPE (평균 절대 백분율 오차) - "몇 % 정도 틀리나?"                        │
│   - 오차를 백분율로 표현                                                     │
│   - MAPE 1% = 예측이 평균 1% 정도 빗나감                                    │
│   - 예: 실제 16%인데 15.84%~16.16% 사이로 예측                              │
│                                                                             │
│ 🎯 신뢰도 - "이 예측을 얼마나 믿을 수 있나?"                                 │
│   - 신뢰도 = 100% - MAPE                                                    │
│   - 신뢰도 98% = 예측이 98% 정확함                                          │
│   - 95% 이상이면 "매우 신뢰할 수 있음"                                       │
│                                                                             │
└─────────────────────────────────────────────────────────────────────────────┘

2. Linear Regression 함수식 (시뮬레이터 사용 방식)
================================================================================

   [GS 충전기 예측 모델]
   ┌─────────────────────────────────────────────────────────────────────────┐
   │ GS충전기(t) = 107.76 × t + 72,328.91                                    │
   │                                                                         │
   │ 쉬운 해석: "GS차지비는 매월 평균 108대씩 충전기를 늘리고 있다"           │
   │                                                                         │
   │ - R²: 0.5397 (54% 설명력 - 변동이 좀 있음)                               │
   │ - MAE: 276.91대 (평균 277대 정도 오차)                                   │
   └─────────────────────────────────────────────────────────────────────────┘

   [시장 전체 예측 모델]
   ┌─────────────────────────────────────────────────────────────────────────┐
   │ 시장전체(t) = 5,587.55 × t + 402,093.96                                 │
   │                                                                         │
   │ 쉬운 해석: "전체 충전기 시장은 매월 평균 5,588대씩 성장하고 있다"        │
   │                                                                         │
   │ - R²: 0.9771 (98% 설명력 - 매우 안정적인 성장)                           │
   │ - MAE: 2,607.18대 (평균 2,607대 정도 오차)                               │
   └─────────────────────────────────────────────────────────────────────────┘

   [점유율 계산 - Ratio 방식 (시뮬레이터 사용)]
   ┌─────────────────────────────────────────────────────────────────────────┐
   │ 점유율(t) = GS충전기(t) ÷ 시장전체(t) × 100                             │
   │                                                                         │
   │ 쉬운 해석: "GS 충전기 수를 시장 전체로 나눠서 점유율 계산"               │
   │                                                                         │
   │ - R²: 0.9589 (96% 설명력 - 매우 좋음!)                                   │
   │ - MAE: 0.1298%p (평균 0.13%p 오차 - 매우 정확!)                          │
   │ - MAPE: 0.78% (100번 예측하면 99번 이상 정확)                            │
   └─────────────────────────────────────────────────────────────────────────┘

   [점유율 직접 예측 - Direct 방식 (비교용)]
   ┌─────────────────────────────────────────────────────────────────────────┐
   │ 점유율(t) = -0.1954 × t + 17.95                                         │
   │                                                                         │
   │ 쉬운 해석: "점유율이 매월 0.2%p씩 감소하는 추세"                         │
   │                                                                         │
   │ - R²: 0.9466 (95% 설명력 - 좋음)                                         │
   │ - MAE: 0.1473%p (Ratio보다 13% 더 큰 오차)                               │
   │ - MAPE: 0.88% (Ratio보다 덜 정확)                                        │
   └─────────────────────────────────────────────────────────────────────────┘
